# 词典与篇目导入 JSON 格式

## 词典 JSON（两种示例）
- 数组：
```json
[
  {"word":"陋室","explain":"简陋的屋子"},
  {"head":"惟","gloss":"只、唯"}
]
```
- 映射：
```json
{
  "德馨": "美德芳香，比喻品德高尚",
  "仙": "指仙人，泛指高人"
}
```

## 篇目 JSON
```json
{
  "title": "陋室铭",
  "author": "刘禹锡",
  "paragraphs": ["山不在高，有仙则名。"],
  "notes": ["……"],
  "translation": "……"
}
```
